Cómo Instalar

# 1. Descarga el plugin

git clone https://github.com/tuusuario/amz-top-products.git

# 2. Sube la carpeta a tu sitio WordPress

Sube la carpeta 'AMZ-Top_products' a /wp-content/plugins/

# 3. Activa el plugin

Ve a Plugins > Plugins instalados y activa 'AMZ Top Products'
